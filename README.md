Chat app for discussion. Invite your collega to join your discussion room and delete room when its done.

### Setup
1. Copy env.exampe to .env
2. Fill the config
3. Run `go get ./...`

### Run The App
`go run ./cmd/todo/main.go`

### Test The App
`go test -v ./...`

### Build The App
`go build -o ./bin/discuss ./cmd/discuss/...`
